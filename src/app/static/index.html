Shed Of Graphs

In the form below you can submit a prompt to generate and filter a set of graphs. Here is what each field does:

  • Order: The number of nodes in the graph.
  • Filter: A filter to apply to the generated graphs. The filter is a string that can contain the following:
    • The maximum number of nodes with a certain degree. For example, "Maximum 2 vertices with degree 2 or 3".
    • The minimum number of nodes with a certain degree. For example, "Minimum 3 vertices with degree 4".
    • The exact number of nodes with a certain degree. For example, "Exactly 4 vertices with degree 3".
    • Only graphs where very node has a certain degree. For example, "Only vertices with degree 3".
  • Minimum Degree: The minimum degree of the nodes in the graph. (3 by default, low numbers combined with high orders greatly affect performance)
